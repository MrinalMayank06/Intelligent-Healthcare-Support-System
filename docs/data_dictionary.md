# Data Dictionary - Intelligent Healthcare Support System

## Main target

- `triage_level`: prediction label used by the ML model.

## Main features

- `age`
- `primary_symptom`
- `fever`
- `oxygen_saturation`
- `heart_rate`
- `pain_level`
- `symptom_days`
- `chronic_condition`

## Data zones

- Raw: original source records and event logs.
- Staged: cleaned and standardized features.
- Curated: model-ready and dashboard-ready facts.
