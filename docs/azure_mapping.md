# Azure Mapping - Intelligent Healthcare Support System

| Requirement | Azure component |
|---|---|
| API hosting | Azure App Service |
| LLM responses | Azure OpenAI / Azure AI Foundry |
| Document/vector search | Azure AI Search or local retrieval replacement |
| Operational database | Cosmos DB Mongo API or MongoDB Atlas |
| Data engineering | ADF, Fabric Lakehouse or Databricks |
| Dashboard | Power BI Service |
| Secrets | App Settings or Key Vault references |

## Environment variables

Use `.env.example` locally and Azure App Settings in cloud.
