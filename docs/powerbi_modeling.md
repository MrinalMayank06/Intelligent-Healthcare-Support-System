# Power BI Modeling - Intelligent Healthcare Support System

## Recommended tables

- Main curated dataset: `data/curated/healthcare_triage_dataset.csv`
- Dashboard fact table: `data/curated/patient_dashboard_facts.csv`
- KPI export: `powerbi/kpi_summary.csv`
- Request log: `artifacts/monitoring/agent_model_request_log.jsonl`

## Recommended measures

- Total records
- High-risk records
- High-risk percentage
- Average risk score
- Model request count
- Agent request count

## Suggested report pages

- Triage distribution
- High-risk symptom groups
- Oxygen and heart-rate trend
- Age-band risk
- Agent support logs
